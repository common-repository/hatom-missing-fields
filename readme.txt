This plugin modify the protocol of all links by its protocol-less counterparts.

== Description ==

Modifies html attributes to add updated value to class.

== Installation ==

1. Install "hAtom Missing Fields" either via the WordPress.org plugin directory, or by uploading the files to your server inside the wp-content/plugins folder of your WordPress installation.
2. Activate "hAtom Missing Fields" plugin via WordPress Settings.
3. It's done. Easy, isn't it?
